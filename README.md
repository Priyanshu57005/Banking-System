# ViaBank - Banking System Backend

ViaBank is a secure and scalable banking backend application built using Node.js, Express.js, and MongoDB. The project follows real-world banking principles including authentication, account management, transaction processing, ledger-based accounting, idempotency validation, email notifications, and secure API design.

## Features

* User Registration & Login
* JWT Authentication
* Password Hashing using bcrypt
* Email Notifications with Nodemailer
* Account Management
* Transaction Processing
* Ledger-Based Accounting System
* Idempotent Transaction Validation
* Account Balance Calculation
* Transaction History
* Account Status Verification
* Blacklist Token Management (Logout)
* Secure REST APIs
* MongoDB Atlas Integration

## Tech Stack

* Node.js
* Express.js
* MongoDB & Mongoose
* JWT Authentication
* bcryptjs
* Nodemailer
* Cookie Parser
* REST APIs

## Project Goal

The goal of ViaBank is to simulate how modern digital banking systems manage accounts, balances, transactions, and security while following backend engineering best practices.
